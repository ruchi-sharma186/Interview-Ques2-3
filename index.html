Q1.what is box sizing?

The box-sizing property determines how the total width and height of an element are calculated.
content-box (default): Width and height include only content; padding and border are added separately.
border-box: Width and height include content, padding, and border, preventing extra growth of the element.

Q2. What is animation-delay?

The animation-delay property specifies the time to wait before starting an animation.

Example: div { animation: fadeIn 2s; animation-delay: 1s; /* Starts after 1 second */ }

Q3. Which property is used to merge cells vertically in CSS Grid?

Use grid-row to span multiple rows.

Example:
.item { grid-row: span 2; /* Merges 2 rows */ }

Q4. Which property is used to merge cells horizontally in CSS Grid?

Use grid-column to span multiple columns.

Example:
.item { grid-column: span 3; /* Merges 3 columns */ }

Q5.Explain CSS Box Model

The CSS Box Model consists of:
Content: The actual text or image inside the element.
Padding: Space between content and border.
Border: The edge around the padding.
Margin: Space outside the border separating elements.

Q1. Difference Between relative and absolute Position

relative: Moves the element relative to its original position while keeping space occupied.

absolute: Moves the element relative to its nearest positioned ancestor (or the viewport if none exists) and removes it from the normal flow.
